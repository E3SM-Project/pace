About

The Energy Exascale Earth System Model (E3SM) is a earth system model that utilizes advanced supercomputers to address problems of national interest.
PACE was designed to visualize E3SM's performance data with ease.

How It Works


Our server first receives the experiment through a client-side program. We use this to retreive basic information about the experiment, which is then moved to PACE'S database. When stored, it can be queried by date,user,machine, etc.


We also read the process tree, which was generated while the experiment ran. This is processed into a readble format suitable for display this data over our website (or any other potential applications). This data can be used for comparing other experiments on this website.
