About

PACE is a framework designed to aggregate and summarize performance data collected from scientific applications like E3SM to derive insights and make them accessible through a web portal.
The Energy Exascale Earth System Model (E3SM) is a high-resolution coupled Earth system model that is designed to address energy-related science challenges while efficiently utilizing Department of Energy (DOE) supercomputers.
PACE enables climate scientists to view executive summary of E3SM experiments' performance and interactively deep-dive as desired.

Architecture


Domain or computer scientists export experiment data from a computational platform (e.g., Summit) to PACE using an upload tool. PACE processes and ingests that information into the backend database. Finally, an user can interactively explore the performance data through the web portal.

Visualization Pipeline


Upon selection of a specific experiment, PACE reads the detailed timing profile to generate the process tree in a format suitable for visualization via the web portal.
